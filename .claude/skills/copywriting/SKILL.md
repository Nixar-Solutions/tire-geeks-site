# Copywriting

Write marketing copy that converts by being specific, customer-language-aligned, and benefit-led — not feature-led, not generic, not AI-flavored.

## Always read first

Before writing any copy:
1. marketing-context.md (Section 6 differentiation, Section 9 customer language, Section 10 voice profile, Section 11 style guide, Section 12 proof points)
2. The actual page being written for and the pages it links to/from
3. Apply the brand-voice skill's Hard Bans without exception

## Page copy structure

Hero: clear value prop in customer language + primary CTA
Subhero: supporting context + secondary CTA  
Body: benefit-led sections with proof points (numbers, testimonials, specifics)
FAQ: real customer questions answered honestly
Closing: restate value + primary CTA

## Rules

- Specifics beat adjectives. "$79 alignment" beats "affordable alignment".
- Benefits beat features. "Same-day install" beats "fast service".
- Customer language beats marketing speak. Use words customers use.
- Proof points beat claims. Cite numbers, testimonials, sources.
- Direct claims beat hedged claims. No "should", "probably", "might".
- One primary CTA per page. Don't dilute.

## Hard bans

Inherit all brand-voice Hard Bans. Plus:
- "Best in class", "industry-leading", "premium quality" (filler)
- "Cutting-edge", "state-of-the-art" (filler)
- "Solution" as a generic noun
- "Utilize" instead of "use"
- "Leverage" instead of "use"
