# Brand Voice

Build a durable voice profile from real source material, then use that profile everywhere instead of re-deriving style or defaulting to generic AI copy.

## Source priority

1. Recent original site copy and blog posts
2. Articles, essays, launch notes
3. Real outbound communications
4. Product docs, README framing

Do NOT use generic platform exemplars as source material.

## What to extract

- Rhythm and sentence length
- Compression vs explanation
- Capitalization norms
- Parenthetical use
- Question frequency and purpose
- How sharply claims are made
- How often numbers, mechanisms, or receipts show up
- Transitions
- What the author never does

## Output

Produce a reusable VOICE PROFILE block downstream skills can consume directly.

## Hard Bans

Delete and rewrite if found:
- Fake curiosity hooks
- "Not X, just Y" framing
- "No fluff"
- Forced lowercase
- LinkedIn thought-leader cadence
- Bait questions
- "Excited to share" / "Thrilled to announce"
- Generic founder-journey filler
- Corny parentheticals
- Soft hedges (should, probably, might) when claims should be direct
