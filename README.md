# Tire Geeks — Sacramento's Premier Tire & Wheel Shop

Modern dark-themed automotive website built with Next.js, Tailwind CSS, GSAP, and Lenis.

## Tech Stack
- **Framework**: Next.js 16+ (App Router, TypeScript)
- **Styling**: Tailwind CSS v4
- **Animations**: GSAP + ScrollTrigger + SplitType + Lenis
- **Icons**: Lucide React + Phosphor Icons
- **Carousel**: Swiper.js
- **Fonts**: Bebas Neue, Oswald, Barlow Condensed, Inter (Google Fonts)
- **Hosting**: Netlify (SSR via @netlify/plugin-nextjs)

## Getting Started
```bash
npm install
npm run dev       # Dev server at localhost:3000
npm run build     # Production build
npm run start     # Production server locally
```

## Deploy
```bash
npm run build
npx netlify deploy --prod
```
